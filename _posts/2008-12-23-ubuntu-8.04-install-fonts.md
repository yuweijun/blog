将字体复制到truetype目录中：

{% highlight bash %}
$> sudo mv Desktop/droids /usr/share/fonts/truetype/droids
{% endhighlight %}

然后更新字体缓存：

{% highlight bash %}
$> sudo fc-cache
{% endhighlight %}
