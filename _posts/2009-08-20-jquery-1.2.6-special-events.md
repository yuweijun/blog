Added in jQuery-1.2.2, can create whole shadow event system.

New events: `mouseenter`, `mouseleave`, `ready`.

{% highlight javascript %}
$("li").bind("mouseenter", function(){
    $(this).addClass("hover");
}).bind("mouseleave", function(){
    $(this).removeClass("hover");
});
{% endhighlight %}
