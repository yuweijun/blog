[Remmina](http://remmina.sourceforge.net/)具备更多的设置项，比如颜色及画面质量、支持ssh tunnel、保存密码等等，确实比现有的默认远程管理工具要更加好用。

{% highlight bash %}
$> sudo apt-get install remmina
{% endhighlight %}
