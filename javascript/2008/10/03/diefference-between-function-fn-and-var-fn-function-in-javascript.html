javascript function的二种定义方式的区别

其中需要说明一下var func = function(){}与function func(){}这二者的区别是后者用function语句定义的variable会先于此function运行前被初始化，因此h()函数调用可以写在其定义语句之前，而前者用var声明的variable则只能在此变量声明之后才可以被调用，与其他用var声明的变量完全一样。
